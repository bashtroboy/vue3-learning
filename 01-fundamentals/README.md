# Fundamentals

Learning Vue 3 basics through practical examples.

## Apps in This Section

### 1. Hello Vue - Basic Reactivity (`01-hello-vue/`)
**Concepts:** Vue instance, reactivity, v-model, methods
**What we'll build:** A simple Content Server object info display

### 2. Components Basics (`02-components/`)
**Concepts:** Creating components, props, emitting events
**What we'll build:** Reusable card components for displaying CS metadata

### 3. Lists and Conditionals (`03-lists-conditionals/`)
**Concepts:** v-for, v-if, v-show, computed properties
**What we'll build:** A list of CS nodes with filtering

### 4. Forms and Input (`04-forms/`)
**Concepts:** Form handling, v-model variations, validation
**What we'll build:** A simple search interface for CS objects

## Key Concepts Covered

- **Reactivity**: Vue's core feature - when data changes, the UI updates automatically
- **Directives**: Special attributes like v-if, v-for, v-model
- **Methods**: Functions that respond to user actions
- **Computed Properties**: Derived data that updates automatically
- **Components**: Reusable pieces of UI
- **Props**: Passing data to child components
- **Events**: Child components communicating back to parents

## Learning Approach

Each app builds on the previous one. We'll:
1. Start with inline Vue (CDN) to focus on concepts
2. Gradually introduce build tools
3. Keep examples practical and relevant to Content Server work
4. Add notes about what's different from Vue 2

Start with `01-hello-vue/` and work through in order!
